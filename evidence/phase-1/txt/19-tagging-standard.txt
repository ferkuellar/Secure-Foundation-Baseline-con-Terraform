# Tagging Standard

## Objective

Ensure resources are traceable, operable, and attributable for cost and governance purposes.

## Mandatory Tags

- Project
- Environment
- Owner
- CostCenter
- BusinessUnit
- ManagedBy
- Repository

## Example

Project      = terraform-secure-foundation
Environment  = dev
Owner        = Fernando
CostCenter   = LAB-001
BusinessUnit = Architecture
ManagedBy    = terraform
Repository   = terraform-secure-foundation

## Principles

- every managed resource must inherit the common tag baseline
- tags must be defined centrally in the environment root
- module-specific tags may be merged only as additions, not replacements
- cost and governance tags are not optional
