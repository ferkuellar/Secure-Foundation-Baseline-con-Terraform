# Bootstrap Runbook

## Purpose

Prepare the repository and workstation for phased Terraform delivery.

## Steps

1. Create repository structure
2. Create base documentation files
3. Initialize git
4. Validate Terraform, Git, PowerShell, and cloud CLI
5. Store outputs in `evidence/phase-0/txt`
6. Capture screenshots for the main milestones
7. Commit the initialized repository

## Evidence

- project tree
- git initialization
- tool versions
- cloud authentication/identity
- base documentation files
